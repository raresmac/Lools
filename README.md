# Lools
🎮 LOOLS | League of Legends Dashboard
LOOLS is a high-performance, real-time strategic dashboard designed for competitive League of Legends play. Built with Blazor WebAssembly and Tailwind CSS, it provides players with a premium interface to track critical in-game data, specifically focusing on enemy summoner spell cooldowns.

✨ Key Features
Dynamic Summoner Tracking: Real-time countdowns for all enemy summoner spells.
Intelligent Haste Calculations: Automatically adjusts cooldowns based on Cosmic Insight and Ionian Boots of Lucidity.
Riot Games Integration: Fetches high-definition champion splash art and spell icons directly from the Data Dragon CDN.
Searchable Roster: Integrated searchable dropdown featuring the full League of Legends roster (up to 2025 releases).
Premium UI/UX: A sleek, "gaming-first" aesthetic with a responsive layout, glowing hover states, and fluid animations.
